About Me

Halo! Perkenalkan, saya Abdurahman, seorang Front End Developer yang baru belajar. Saya ingin berbagi sedikit tentang diri saya dan perjalanan belajar saya dalam dunia pengembangan web. Saya selalu tertarik dengan dunia teknologi, terutama bagaimana desain dan interaksi antarmuka yg dapat membuat pengalaman pengguna menjadi lebih baik. Itulah mengapa saya memutuskan untuk belajar menjadi seorang Front End Developer. Perjalanan saya dalam belajar Front End Development dimulai ketika saya mulai menggali berbagai sumber belajar online. Saya memulai dengan dasar-dasar HTML, CSS, dan JavaScript. Awalnya, saya merasa sedikit kewalahan dengan banyaknya informasi dan teknologi yang perlu dipelajari. Namun, semakin lama saya belajar, semakin saya merasa terinspirasi untuk terus maju. Saat ini, saya memiliki pemahaman yang lebih baik tentang HTML dan CSS, dan saya terus memperdalam pengetahuan saya dalam JavaScript. Hal ini saya dapati dari Hacktiv8, yg mana mereka mengajarkan saya kursus tentang Front End Developer dengan baik. Selama perjalanan belajar saya, saya menemukan bahwa pengembangan Front End bukan hanya tentang menulis kode, tetapi juga tentang memahami kebutuhan pengguna dan menciptakan pengalaman yang intuitif dan menarik. Saya senang bekerja dalam tim dan berkolaborasi dengan desainer dan pengembang Back End untuk menciptakan produk yang hebat. Saat ini, saya masih dalam tahap belajar dan ingin terus meningkatkan kemampuan saya dalam pengembangan Front End. Saya juga tertarik untuk terus mengikuti perkembangan teknologi web yang terus berubah. Terima kasih telah membaca. Saya sangat antusias untuk terus belajar dan berkembang dalam peran sebagai seorang Front End Developer. Jika Anda ingin berbicara lebih lanjut atau berkolaborasi dalam proyek web, jangan ragu untuk menghubungi saya. Saya sangat senang dapat berkenalan dengan sesama pengembang web atau siapa pun yang memiliki minat dalam teknologi.
